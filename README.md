# EtherDrops Miniapp

### Fitur
- Auto complete task
- Daily check in
- Multi Akun


### Step

**1. clone repo dan masuk ke folder**
```
git clone https://github.com/yogik23/skw-etherdrop && cd skw-etherdrop
```

**2. Install Module**
```
npm install
```

**3. submit query di file** `data.txt`
```
nano data.txt
```
format query 
```
query_id=AAE1 
query_id=AAE2 
query_id=AAE3
```

**4. Jalankan bot**
```
node main.js
```

**Sodah kerjekan mun sian botnye**
